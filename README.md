# TempServ
A velocity plugin to add temporary servers

# Building

Run ./gradlew build 
Your jar will be in build/libs

You can download the CI Build from GitHub Actions in Artifacts.

For Devs:

```
Maven repository
	
Name  tempserv
URL   https://packages.hexus.eu/repository/HXMC-SNAPSHOT/
```
```
Dependencies
```
